1. Title: Stock Price Modeling Using AR & MA Models

Problem Statement:

Traditional trend-based analysis fails to capture the complex temporal dependencies present in financial time series data, making short-term stock price prediction difficult. A financial analytics firm requires sophisticated models to analyze stock price behavior to support trading and investment strategies.

Aim:

* Transforming non-stationary financial data into stationary time series.
* Utilizing statistical plots to identify optimal lag parameters ($p$ and $q$).
* Building, training, and diagnosing ARIMA models to ensure they capture temporal dependencies without overfitting.
* Establishing a validated, serialized model (.pkl) ready for integration into a professional trading or investment strategy pipeline.

Project Overview:

* This project focuses on modeling short-term stock price behavior using classic time series analysis techniques: Autoregressive (AR) and Moving Average (MA) models. The objective is to understand temporal dependencies in financial data and build reliable forecasting models.

* The project involves preprocessing non-stationary price data, identifying optimal model parameters using ACF and PACF plots, fitting models using `statsmodels`, evaluating performance, and exporting the trained model for production use.

Objectives:

  • Data Acquisition: Fetch historical stock data using yfinance.
  • Exploratory Data Analysis (EDA): Visualize stock price trends and analyze stationarity.
  • Preprocessing: Difference the data to achieve stationarity.
  • Model Identification: Use ACF and PACF plots to determine optimal lag parameters ($p$ for AR, $q$ for MA).
  • Model Building: Fit AR, MA, and combined ARMA models.
  • Diagnostics: Analyze residuals to ensure they are white noise.
  • Performance Evaluation: Compare models using AIC (Akaike Information Criterion) and BIC (Bayesian Information Criterion).

  1. Stationarity Check & Transformation:

    Time series models like AR and MA require the data to be stationary (mean and variance do not change over time). Stock prices are rarely stationary.
In[23]:
# Augmented Dickey-Fuller Test
def check_stationarity(timeseries):
    result = adfuller(timeseries)
    print('ADF Statistic:', result[0])
    print('p-value:', result[1])
    if result[1] <= 0.05:
        print("Data is stationary")
    else:
        print("Data is non-stationary")

check_stationarity(df['Close'])

# Transformation: Differencing to achieve stationarity
df_diff = df.diff().dropna()
print("\nAfter Differencing:")
check_stationarity(df_diff['Close'])
